## 1. 짧게, 하지만 얕지 않게

> 간결은 자신감이다. 장황은 두려움이다.(Axios 스타트업)

스마트 브레비티: 더 분명하게 생각하고, 명확하게 커뮤니케이션하여 자신과 다른 사람의 시간을 줄이는 체계이자 전략.

> 훨씬 더 적은 단어로 훨씬 더 많이 전달하라

## 2. 스마트 브레비티 해설

핵심 네가지

1. 힘 있는 "도발"

- 짧고 강력한 단어. ex) 당신의 CEO는 점심 친구가 없다.

2. 강력한 첫 문장, "리드"

- 독자가 모른는 것, 알고 싶은 것, 알아야 할 것을 이야기하자. 직설적이고 짧고 날카롭게. ex) 사무실이 다시 문을 열었지만, 일하러 가고 싶은 건 임원뿐이다.

3. 맥락, "왜 중요한가"

- 모르는걸 질문하는건 부끄럽기에, 왜 중요한지 설명해줘야한다.

4. 더 알아볼 사람을 위한, "깊이 알아보기"

- 사람들이 원하는 것 이상으로 읽거나 듣도록 강요하지 마라.

```
나는 사실 줄글형태도 좋아한다. 진짜 글 잘 쓰는 사람은 글이 길어도 스토리에 빠져들게 하는 힘이 있다.
그건 더 높은 단계의 스킬이고 요약된 bullet 형태 예시를 보니 확실히 그냥 줄글보다는 잘 읽힌다.

```
